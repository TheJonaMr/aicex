# aicex
Files for Advanced Integrated Circuits


## Requirements

- ngspice > 34

- python > 3.8


## Getting Started

Quick install 

``` sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/wulffern/aicex/main/install.sh)"
```

Check that test runs

``` sh
cd aicex
make test
```
